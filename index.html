O que é uma Função Afim?

Uma função afim (também chamada de função de 1º grau) é qualquer função que pode ser escrita na forma:

f(x) = ax + b
onde a ≠ 0 e a, b são números reais
a
Coeficiente angular
Indica a inclinação da reta. É a taxa de variação da função — quanto f(x) varia quando x aumenta 1 unidade.
b
Coeficiente linear
É o valor de f(0). Indica o ponto onde a reta cruza o eixo y (eixo das ordenadas).

Propriedades importantes

PropriedadeValor / Explicação
DomínioTodos os reais: D = ℝ
ImagemTodos os reais: Im = ℝ
GráficoUma reta — nunca curva, nunca vertical
Zero da funçãoValor de x tal que f(x) = 0, ou seja: x = –b/a
Taxa de variaçãoΔy/Δx = a (sempre constante)

O que o coeficiente angular a nos diz?

a > 0 Função crescente — quanto maior x, maior f(x). A reta sobe da esquerda para a direita.
a < 0 Função decrescente — quanto maior x, menor f(x). A reta desce da esquerda para a direita.
|a| maior Reta mais inclinada (mais "íngreme").

Casos especiais da função afim

1. Função Linear (b = 0)

f(x) = ax

Passa obrigatoriamente pela origem (0, 0). O coeficiente linear é zero, por isso a reta sempre começa no centro do plano cartesiano.

Exemplo: f(x) = 3x → f(0) = 0, f(1) = 3, f(2) = 6

2. Função Identidade (a = 1, b = 0)

f(x) = x

A imagem sempre é igual ao domínio: f(1) = 1, f(2) = 2, f(–3) = –3. Forma a bissetriz do 1º e 3º quadrantes (ângulo de 45°).

3. Função Constante (a = 0) — caso limite

f(x) = b

Atenção: tecnicamente, quando a = 0 a função não é afim (pois exigimos a ≠ 0), mas é um caso limite importante. O gráfico é uma reta horizontal.

Exemplo: f(x) = 4 → sempre vale 4, independente de x.

Comparando as retas no mesmo gráfico

f(x) = 2x + 3 f(x) = x (identidade) f(x) = –x + 2 f(x) = 4 (constante)

Exemplo 1 — Problema do táxi 🚕

Um táxi cobra R$ 5,00 de bandeirada mais R$ 2,50 por km rodado. Qual é a função que representa o preço da corrida?

f(x) = 2,5x + 5
x = quilômetros rodados | f(x) = preço em reais
1
a = 2,5 → a cada km percorrido, o preço sobe R$ 2,50
2
b = 5 → mesmo sem sair do lugar, já paga R$ 5,00 (bandeirada)
3
Quanto custa 10 km? f(10) = 2,5 × 10 + 5 = 25 + 5 = R$ 30,00
4
Zero da função: 2,5x + 5 = 0 → x = –2. Sem sentido prático aqui (km não pode ser negativo).

Exemplo 2 — Encontrar a lei pelos pontos 📍

Uma função afim passa pelos pontos (2, 7) e (4, 11). Qual é a lei?

1
Calcule o coeficiente angular:
a = (y₂ – y₁) / (x₂ – x₁) = (11 – 7) / (4 – 2) = 4 / 2 = 2
2
Calcule o coeficiente linear:
Use o ponto (2, 7): 7 = 2 × 2 + b → 7 = 4 + b → b = 3
3
Verifique com o outro ponto:
f(4) = 2 × 4 + 3 = 11 ✓
f(x) = 2x + 3

Exemplo 3 — Encontrar o zero da função 🎯

Para f(x) = –3x + 9, encontre o zero da função e interprete graficamente.

1
Zero significa f(x) = 0:
–3x + 9 = 0
2
Isole o x:
–3x = –9 → x = –9/(–3) → x = 3
3
Interpretação gráfica: O gráfico corta o eixo x exatamente em x = 3. Para x < 3, f(x) > 0; para x > 3, f(x) < 0.
4
Verifique: f(3) = –3 × 3 + 9 = –9 + 9 = 0 ✓
